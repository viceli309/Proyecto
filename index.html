¿QUE ES BRAIN TEST?

BRAIN TEST te permitira crear test pudiendo editar cada pregunta con elementos basicos y te permitira el uso de contenido multimedia(imagenes o videos) para apoyarte como uso a la hora de realizar las preguntas.

CREA TUS TEST

BRAIN TEST puedes crear tus test de forma simple y sencilla apoyandote en la tematica que gustes aunque principalmente el sitio esta diseñado para las ciencias naturales queremos fomentar ese aprendizaje.

RESUELVE TEST

BRAIN TEST resuelve los test que los usuario hayan creado, solo solicita el identificador del test que planeas resolver.
